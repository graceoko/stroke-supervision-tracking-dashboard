# Dashboards & Visual Outputs

This folder contains visual outputs and dashboard artifacts developed to support stroke encounter monitoring, guideline compliance assessment, and quality improvement reporting.

## Power BI Dashboard
- `powerbi_dashboard_overview.jpg` provides a high-level view of stroke encounters and compliance trends.
- `OHMC_dashboard.pbix` is the Power BI source file used to generate the dashboard.

The dashboard is designed for clinical staff, compliance teams, and leadership to support data-driven decision-making.

## Microsoft Access Interface & Queries
Screenshots of the Microsoft Access front-end and queries used to support reporting:
- Main menu and patient data entry forms
- Compliance summary queries
- Non-compliance query results used for monitoring guideline adherence

These visuals demonstrate how data flows from structured entry and querying into consolidated dashboard reporting.
